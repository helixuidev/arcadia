# Arcadia.Core

Core foundation for [Arcadia Controls](https://arcadiaui.com) — base classes, utilities, theming abstractions, and accessibility helpers for Blazor.

## What's Included

- **HelixComponentBase** — base class with Class, Style, AdditionalAttributes
- **HelixInputBase\<T\>** — base for form inputs with Value/ValueChanged/ValueExpression
- **CssBuilder / StyleBuilder** — fluent CSS class and inline style construction
- **IdGenerator** — unique IDs for accessibility (aria-describedby, label-for)
- **FocusTrap** — keyboard focus containment for modals/dialogs
- **LiveRegion** — ARIA live region for screen reader announcements
- **AriaHelper / KeyboardNavigation** — accessibility utilities

## Installation

```
dotnet add package Arcadia.Core
```

This package is automatically included when you install any Arcadia component package.
